# Area 对象

Area 对象代表图像映射的一个区域（图像映射指的是带有可点击区域的图像）

在 HTML 文档中 `<area>` 标签每出现一次，就会创建一个 Area 对象。

## Area 对象属性

| 属性                | 描述                                           |
| :------------------ | :--------------------------------------------- |
| areaObject.alt      | 设置或返回当浏览器无法显示某个区域时的替换文字 |
| areaObject.coords   | 设置或返回图像映射中可点击区域的坐标           |
| areaObject.hash     | 设置或返回某个区域中的 URL 的锚部分            |
| areaObject.host     | 设置或返回个区域中 URL 的主机名和端口          |
| areaObject.hostname | 设置或返回 href 属性值得主机部分               |
| areaObject.href     | 设置或返回某个区域中 href 属性值               |
| areaObject.noHref   | 设置或返回某个区域的 nohref 属性值             |
| areaObject.pathname | 设置或返回某个区域 href 属性值的路径名部分     |
| areaObject.port     | 设置或返回某个区域 href 属性值的端口部分       |
| areaObject.protocol | 设置或返回某个区域 href 属性值的协议部分       |
| areaObject.search   | 设置或返回某个区域 href 属性值的查询字符串部分 |
| areaObject.shape    | 设置或返回某个区域 shape 属性值                |
| areaObject.target   | 设置或返回某个区域 target 属性值               |
