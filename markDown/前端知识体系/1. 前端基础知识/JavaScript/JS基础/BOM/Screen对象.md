# Screen 对象

`window.screen` 对象包含用户屏幕的信息。

## Screen 对象属性

| 属性        | 说明                                     |
| :---------- | :--------------------------------------- |
| availHeight | 返回屏幕的高度（不包括 Windows 任务栏）  |
| availWidth  | 返回屏幕的宽度（不包括 Windows 任务栏）  |
| colorDepth  | 返回目标设备或缓冲器上的调色板的比特深度 |
| height      | 返回屏幕的总高度                         |
| pixelDepth  | 返回屏幕的颜色分辨率（每象素的位数）     |
| width       | 返回屏幕的总宽度                         |
