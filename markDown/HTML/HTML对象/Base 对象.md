# Base 对象

Base 对象代表 HTML 的 base 元素。

base 元素适用页面中于指定的默认地址或者所有链接的目标地址。

在 HTML 文档中 `<base>` 每出现一次，Base 对象就会被创建。

## Base 对象属性

| 属性   | 描述                                       |
| :----- | :----------------------------------------- |
| href   | 设置或返回针对页面中所有链接的基准 URL     |
| target | 设置或返回针对页面中所有链接的默认目标框架 |
